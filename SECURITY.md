# Security Policy

## Supported Versions

Security updates are provided for the latest release.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public issues.

Report security issues privately to the maintainers.

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fixes if available

Please allow maintainers reasonable time to investigate before public disclosure.

## Security Scope

Security reports may include:

- Authentication issues
- Encryption vulnerabilities
- Remote code execution
- Data exposure
- Network protocol vulnerabilities
- Server security problems

## Current Security Status

FOSSil Chat is currently in early development.

Security features such as end-to-end encryption and advanced identity verification are planned but not yet implemented.
