## [v0.1.0] - 2026-07-16

### Added

- Initial FOSSil Chat release
- Rust client-server architecture
- Native desktop client built with Dioxus
- Tokio asynchronous TCP server
- Shared Rust library for communication structures
- JSON packet communication system
- Client join and leave handling
- Real-time message broadcasting
- Username validation
- Message identifiers and timestamps
- Automated test suite
- Linux x86_64 binaries
- Windows x86_64 binaries
- SHA-256 release verification

### Known Limitations

- No end-to-end encryption
- No user authentication
- No persistent storage
- No federation support
- No file sharing
- Limited client features
