# Code of Conduct

## Our Commitment

FOSSil Chat aims to provide a welcoming, respectful, and collaborative environment for all contributors.

## Expected Behaviour

Contributors should:

- Be respectful and constructive
- Welcome different viewpoints
- Provide helpful feedback
- Focus on improving the project
- Respect other contributors

## Unacceptable Behaviour

The following are not acceptable:

- Harassment or discrimination
- Personal attacks
- Trolling
- Publishing private information
- Deliberately disrupting the project

## Enforcement

Maintainers may remove content, reject contributions, or restrict participation when this code of conduct is violated.

## Contact

For conduct concerns, contact the project maintainers privately.
